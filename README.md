# 🧮 Digital Calculator  https://abul-kalam7.github.io/Online-Calculator/

## ✨ Features
- Addition ➕  
- Subtraction ➖  
- Multiplication ✖️  
- Division ➗  
- Percentage (%)  
- **AC** — Clear all input  
- **C** — Delete last entry
